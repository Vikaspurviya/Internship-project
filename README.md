# 🛒 ShopVista - Advanced E-Commerce Website

ShopVista is a modern, responsive, and interactive e-commerce website built using **HTML, CSS, and JavaScript**. The project provides a smooth online shopping experience with features like product search, wishlist, shopping cart, category filtering, responsive design, and local storage support.

---

## 🚀 Features

### 🏠 Home Page

* Modern responsive design
* Attractive hero section
* Product showcase
* Category navigation

### 🔍 Search Functionality

* Real-time product search
* Instant filtering of products

### 🛒 Shopping Cart

* Add products to cart
* Cart item counter
* Cart sidebar
* Remove items from cart
* Local storage support

### ❤️ Wishlist

* Add products to wishlist
* Remove products from wishlist
* Interactive wishlist icons

### 🎨 User Interface

* Modern product cards
* Hover animations
* Toast notifications
* Mobile-friendly layout
* Responsive design

### 🌙 Advanced Features

* Dark Mode Support
* Hero Slider
* Category Filters
* Local Storage Data Persistence

## 💻 How It Works

### Add To Cart

* User clicks "Add To Cart"
* Product is added to cart
* Cart count updates automatically
* Data is stored in Local Storage

### Wishlist

* User clicks heart icon
* Product is added/removed from wishlist

### Search

* User enters product name
* Matching products are displayed instantly

### Cart Sidebar

* Opens from right side
* Displays selected products
